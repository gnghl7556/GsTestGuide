# GS 인증 시험 가이드 - 프로세스 데이터

이 폴더는 GS 인증 시험의 전체 프로세스 데이터를 마크다운 형식으로 관리합니다.
옵시디언에서 이 폴더를 Vault로 열면 시각적으로 탐색할 수 있습니다.

## 폴더 구조

- `process/` - 7단계 시험 프로세스 정의
- `defect-references/` - 품질특성별 결함 사례 DB
- `templates/` - 양식/템플릿 참조
- `rules/` - 실행 규칙 (게이트 조건 등)
- `theme/` - 카테고리 정의 및 테마 색상

## 프로세스 7단계

1. 시험 준비 (SETUP)
2. 시험 착수
3. 초기 시험
4. 결함 리포트
5. 패치 및 회귀
6. 인증위 심사
7. 시험 종료
