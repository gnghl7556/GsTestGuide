### 알 수 없는 메시지 제공
- **심각도**: M
- **빈도**: A
- **설명**: 특정 기능 실행 시 '500 에러입니다', 'java.io.IOException' 등 사용자가 이해할 수 없는 시스템 오류 메시지가 노출됨. 사용자 친화적인 안내 메시지로 대체 필요.

### 기능 정보 미제공
- **심각도**: M
- **빈도**: A
- **설명**: 프로그램에서 제공하는 기능(예: 정책 설정, 인증 방식 등)에 대한 설명이 매뉴얼이나 프로그램 내 도움말에 제공되지 않음.

### 안내 메시지 미제공
- **심각도**: M
- **빈도**: A
- **설명**: 입력 필드에 유효하지 않은 값(예: 시작일이 종료일보다 이후, 잘못된 형식) 입력 후 기능 실행 시, 형식 확인에 대한 안내 메시지가 제공되어야 하지만 그렇지 않음.

### 필수 항목 미표시
- **심각도**: M
- **빈도**: A
- **설명**: 필수 입력 항목에 대한 표시(예: * 또는 (필수))가 누락되어 있어, 사용자가 어떤 항목을 반드시 입력해야 하는지 알 수 없음.

### 잘못된 기능 정보 제공
- **심각도**: M
- **빈도**: A
- **설명**: 매뉴얼에 실제 프로그램에 존재하지 않는 기능을 제공한다고 잘못 기재되어 있거나, 기능의 동작 방식이 매뉴얼과 다르게 설명됨.

### 잘못된 안내 메시지 제공
- **심각도**: M
- **빈도**: A
- **설명**: 특정 동작(예: 삭제된 사용자 재등록) 시 실제 상황과 맞지 않는 안내 메시지(예: '동일한 사용자가 존재합니다')가 제공되어 사용자 혼란 야기.

### 매뉴얼과 프로그램 불일치
- **심각도**: M
- **빈도**: A
- **설명**: 매뉴얼에 기재된 기능명, 설정값 범위, 동작 방식 등이 실제 프로그램과 일치하지 않음(예: 매뉴얼에는 5 이하 설정이라 했지만 실제로는 초과값 입력 가능).

### 오조작 방지 미제공
- **심각도**: M
- **빈도**: A
- **설명**: 삭제, 초기화 등 되돌릴 수 없는 중요 동작 실행 시, 재확인을 위한 확인 다이얼로그나 경고 메시지가 제공되지 않음.

### 입력필드 오류
- **심각도**: M
- **빈도**: A
- **설명**: 숫자만 입력해야 하는 필드에 음수나 문자가 입력 가능하거나, 날짜 필드에 유효하지 않은 값 입력 시 형식 검증이 동작하지 않음.

### 화면 표시 미흡
- **심각도**: L
- **빈도**: A
- **설명**: 테이블 컬럼명이 잘려서 보이거나, UI 요소가 화면 해상도에 따라 겹치는 등 화면 레이아웃이 미흡함.

### 불필요한 페이지 제공
- **심각도**: M
- **빈도**: A
- **설명**: 필수 입력값 미입력 상태에서 기능 실행 시, 적절한 안내 메시지 대신 서버 오류 페이지(예: 500 Server Error, 404 Not Found)가 표시됨.

### 잘못된 오류 메시지 제공
- **심각도**: M
- **빈도**: A
- **설명**: 파일 형식 오류 등에서 사용자 이해 가능한 메시지가 아닌 시스템 예외(java.io.IOException 등)가 그대로 노출됨.

### 알 수 없는 안내 메시지 제공
- **심각도**: M
- **빈도**: A
- **설명**: 기능 실행 시 사용자가 이해하기 어려운 전문 용어(예: 'A클래스 입니다')가 포함된 안내 메시지가 제공됨.

### 비일관적인 기능명 사용
- **심각도**: M
- **빈도**: A
- **설명**: 동일한 기능이나 개념을 서로 다른 페이지에서 다른 명칭으로 사용함(예: '물리 네트워크'와 '외부 네트워크'가 동일 대상).

### 불필요한 메뉴 제공
- **심각도**: M
- **빈도**: A
- **설명**: 프로그램에 실제로 동작하지 않거나 접근 권한이 없는 메뉴가 표시되어, 클릭 시 오류가 발생하거나 아무 동작도 하지 않음.

### 도움말 미제공
- **심각도**: M
- **빈도**: A
- **설명**: 복잡한 설정이나 전문 용어가 포함된 기능에 대한 도움말(툴팁, 가이드 등)이 제공되지 않음.

### 잘못된 컬럼명 제공
- **심각도**: M
- **빈도**: A
- **설명**: 테이블의 컬럼 헤더가 실제 데이터와 일치하지 않음(예: '정책 설명' 컬럼에 '아이디/이름/부서명' 등이 표시).

### 단위 정보 미제공
- **심각도**: M
- **빈도**: A
- **설명**: 입력 필드 옆에 단위(초, 분, MB, % 등)가 표시되지 않아, 사용자가 어떤 단위로 입력해야 하는지 명확하지 않음.

### 접속시간 제한 표시 오류
- **심각도**: M
- **빈도**: A
- **설명**: 프로그램 창을 최소화했다가 복원 시, 접속 시간 제한이 실제 경과 시간과 맞지 않게 표시됨.

### 표시 오류
- **심각도**: M
- **빈도**: A
- **설명**: 수치, 날짜, 상태값 등이 실제 데이터와 다르게 화면에 표시되거나, 조회 결과가 올바르지 않게 노출됨.
