### 기능 미동작 (버튼/메뉴)
- **심각도**: H
- **빈도**: A
- **설명**: 특정 버튼(예: 삭제, 초기화, 등록) 클릭 시 해당 기능이 전혀 동작하지 않고 아무런 반응이 없음.

### 통계 기능 오류
- **심각도**: H
- **빈도**: A
- **설명**: 통계 조회 기능(접속 통계, 사용량 통계, 모니터링 등) 실행 시, 데이터가 존재함에도 조회 결과가 표시되지 않거나 잘못된 값이 표시됨.

### API 호출 기능 오류
- **심각도**: H
- **빈도**: A
- **설명**: API 호출(인증, 전자서명, 출금동의 등) 시 정상적인 응답이 제공되어야 하지만, 요청이 전달되지 않거나 결과가 반환되지 않음.

### 수정 기능 오류
- **심각도**: M
- **빈도**: A
- **설명**: 데이터 수정 후 저장 시 완료 메시지는 표시되지만, 실제로는 수정이 반영되지 않고 기존 값이 유지됨.

### 조회 기능 오류
- **심각도**: M
- **빈도**: A
- **설명**: 검색/조회 조건 입력 후 실행 시, 조건에 맞는 결과가 표시되지 않거나 전체 데이터가 그대로 표시됨.

### 등록 기능 오류
- **심각도**: H
- **빈도**: A
- **설명**: 필수 정보를 입력하고 등록 버튼 클릭 시, 등록이 완료되지 않거나 오류가 발생함.

### 삭제 기능 오류
- **심각도**: H
- **빈도**: A
- **설명**: 삭제 기능 실행 시 대상이 삭제되지 않거나, 삭제 후에도 목록에 계속 표시됨.

### 할당/배정 기능 오류
- **심각도**: H
- **빈도**: A
- **설명**: 사용자, 리소스, 라이선스 등의 할당/배정 기능이 동작하지 않아 할당 완료 후에도 실제 적용이 안 됨.

### 권한 설정 기능 오류
- **심각도**: H
- **빈도**: A
- **설명**: 사용자 권한 변경, 메뉴 접근 권한 설정 등이 정상 반영되지 않아 변경 후에도 이전 권한으로 동작함.

### 다운로드 기능 오류
- **심각도**: H
- **빈도**: A
- **설명**: 파일 다운로드 버튼 클릭 시 다운로드가 시작되지 않거나, 빈 파일/잘못된 파일이 다운로드됨.

### 인증/로그인 관련 오류
- **심각도**: H
- **빈도**: A
- **설명**: 계정 상태(휴면, 제한 등)에 따른 접근 제어가 동작하지 않아, 제한되어야 할 계정으로 로그인이 가능하거나 인증 절차가 정상 동작하지 않음.

### 정렬/필터 기능 오류
- **심각도**: M
- **빈도**: A
- **설명**: 테이블 정렬 기능 실행 시 정렬이 적용되지 않거나, 필터 조건이 반영되지 않아 전체 데이터가 표시됨.

### 설정값 미적용
- **심각도**: H
- **빈도**: A
- **설명**: 보안 정책, 시스템 설정 등의 값을 변경하고 저장해도 실제 동작에 반영되지 않음(예: 차단 설정 후에도 해당 기능 사용 가능).

### 선택 기능 오류
- **심각도**: H
- **빈도**: A
- **설명**: 팝업창이나 목록에서 항목을 선택해도 선택이 반영되지 않거나 동작하지 않음.

### 초기화 기능 오류
- **심각도**: H
- **빈도**: A
- **설명**: 초기화 버튼 클릭 시 입력 필드나 설정값이 기본값으로 돌아가지 않음.

### 로그아웃 오류
- **심각도**: M
- **빈도**: A
- **설명**: 로그아웃 기능 실행 후에도 세션이 유지되거나, 뒤로 가기로 이전 페이지 접근이 가능함.

### 알림/이력 조회 기능 오류
- **심각도**: H
- **빈도**: A
- **설명**: 이벤트 발생(임계치 초과, 정책 변경 등) 시 알림 이력에 기록되어야 하지만 조회되지 않음.

### 네트워크/서브넷 관리 오류
- **심각도**: H
- **빈도**: A
- **설명**: 네트워크 또는 서브넷의 생성, 수정, 삭제 기능이 정상 동작하지 않음.

### 상태정보 갱신 오류
- **심각도**: M
- **빈도**: A
- **설명**: 상태 정보 요청/갱신 시 최신 데이터가 반영되지 않고 이전 상태가 유지됨.

### 엑셀/CSV 내보내기 오류
- **심각도**: M
- **빈도**: A
- **설명**: 데이터 내보내기 실행 시 일부 데이터가 누락되거나, 파일 형식이 올바르지 않아 열리지 않음.
