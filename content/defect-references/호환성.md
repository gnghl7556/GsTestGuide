### 브라우저 호환성 오류
- **심각도**: M
- **빈도**: A
- **설명**: 특정 웹 브라우저(예: Chrome, Edge, Firefox)에서 프로그램의 UI 레이아웃이 깨지거나 기능이 정상 동작하지 않음.

### 운영체제 호환성 오류
- **심각도**: M
- **빈도**: A
- **설명**: 특정 운영체제 버전(예: Windows 10 vs 11, Android 버전)에서 프로그램이 정상 동작하지 않거나 설치가 불가능함.

### 화면 해상도 호환성 미흡
- **심각도**: L
- **빈도**: A
- **설명**: 특정 화면 해상도나 디스플레이 배율(125%, 150% 등)에서 UI 요소가 겹치거나 잘려서 표시됨.

### 인터페이스 호환성 오류
- **심각도**: M
- **빈도**: A
- **설명**: 외부 시스템(API, DB, 인증 서버 등)과의 연동에서 데이터 형식 불일치나 프로토콜 차이로 인한 통신 오류가 발생함.
