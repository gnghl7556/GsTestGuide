### XSS 취약점 (크로스 사이트 스크립트)
- **심각도**: H
- **빈도**: A
- **설명**: 입력 필드에 스크립트(예: `<script>alert('XSS')</script>`)를 입력하여 저장하면, 해당 페이지 조회 시 스크립트가 실행됨. 프로그램 전반적으로 확인 필요.

### 암호화 되지 않은 개인정보 저장
- **심각도**: H
- **빈도**: A
- **설명**: 사용자의 민감한 개인정보(CI, 비밀번호, 주민번호 등)가 로그 파일이나 DB에 암호화되지 않은 평문 상태로 저장됨.

### 비밀번호 평문 조회
- **심각도**: H
- **빈도**: A
- **설명**: 관리자 화면에서 사용자 비밀번호가 평문으로 조회 가능하거나, 네트워크 전송 시 암호화되지 않음.

### 구 버전 라이브러리 사용 (jQuery)
- **심각도**: M
- **빈도**: A
- **설명**: 보안 취약점이 알려진 구 버전의 jQuery(예: 1.x, 2.x)를 사용 중. 최신 버전으로 업데이트하거나 대체 필요.

### 구 버전 라이브러리 사용 (기타)
- **심각도**: M
- **빈도**: A
- **설명**: Lodash, Moment.js 등 보안 취약점이 알려진 구 버전 라이브러리를 사용 중. CVE 목록 확인 후 업데이트 필요.

### 개인정보 수정 시 재인증 절차 부재
- **심각도**: M
- **빈도**: A
- **설명**: 비밀번호, 이메일, 전화번호 등 중요 개인정보 변경 시, 현재 비밀번호 확인이나 본인인증 절차 없이 변경이 가능함.

### HTTP 보안헤더 누락 (X-Frame-Options)
- **심각도**: M
- **빈도**: A
- **설명**: 클릭재킹 방지를 위한 X-Frame-Options 응답 헤더가 설정되어 있지 않아, iframe을 통한 페이지 삽입 공격에 노출됨.

### HTTP 보안헤더 누락 (X-XSS-Protection)
- **심각도**: M
- **빈도**: A
- **설명**: 브라우저 XSS 필터 활성화를 위한 X-XSS-Protection 응답 헤더가 설정되어 있지 않음.

### 반복된 인증시도 제한 기능 미흡
- **심각도**: M
- **빈도**: A
- **설명**: 로그인 시 비밀번호를 5회 이상 틀려도 계정 잠금이나 지연 시간 등의 제한 조치가 동작하지 않아, 무차별 대입 공격에 취약함.

### 비밀번호 설정 기능 오류
- **심각도**: M
- **빈도**: A
- **설명**: 비밀번호 정책(최소 길이, 특수문자 포함, 대문자 포함 등)이 설정되어 있지만, 정책에 맞지 않는 비밀번호로도 설정이 가능함.

### 실행파일 업로드 방지 오류
- **심각도**: H
- **빈도**: A
- **설명**: 파일 업로드 시 실행 가능한 파일(.exe, .bat, .sh 등)이 차단되지 않아 악성 파일 업로드가 가능함.

### 접근통제 오류
- **심각도**: H
- **빈도**: A
- **설명**: URL 직접 입력이나 파라미터 변조를 통해 권한 없는 페이지나 타인의 데이터에 접근이 가능함.

### SameSite 쿠키 설정 미흡
- **심각도**: M
- **빈도**: A
- **설명**: CSRF 공격 방지를 위해 쿠키의 SameSite 속성이 Strict 또는 Lax로 설정되어야 하나, 설정이 누락되어 있음.

### 비밀번호 입력값 표시 오류
- **심각도**: M
- **빈도**: A
- **설명**: 비밀번호 입력 필드에서 입력값이 마스킹(●●●) 처리되지 않고 평문으로 노출됨.

### 서버 버전 정보 공개
- **심각도**: M
- **빈도**: A
- **설명**: HTTP 응답 헤더에 서버 소프트웨어 버전(IIS, Apache, PHP, ASP.NET 등)이 노출되어, 공격자가 버전별 취약점을 파악할 수 있음.

### 안전하지 않은 TLS/SSL 사용
- **심각도**: M
- **빈도**: A
- **설명**: TLS 1.0, SSL 3.0 등 보안 취약점이 알려진 구 버전 프로토콜이 허용되어 있음. TLS 1.2 이상만 허용 필요.

### 계정 유효성 추측 가능한 에러 메시지
- **심각도**: M
- **빈도**: A
- **설명**: 로그인 실패 시 '아이디가 존재하지 않습니다' 또는 '비밀번호가 틀렸습니다' 등 구체적인 메시지로 계정 존재 여부를 추측할 수 있음.
