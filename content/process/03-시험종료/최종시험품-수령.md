## 설명
업체로부터 최종 시험품을 CD 형태로 수령한다. 시험 결과와 무관하게 필수이다.

## 체크포인트
- [ ] 업체가 최종 시험품을 CD로 제공했는가?
- [ ] CD에 프로그램, 사용자 매뉴얼, 제품설명서가 포함되어 있는가? [ref: 프로그램, 사용자 매뉴얼, 제품 설명서]
- [ ] 최초 제공 시험품과 동일한 형태인가?
- [ ] 프로그램 개발 코드의 해시값이 포함되어 있는가? [ref: 코드 해시 변환값]

## 증빙 예시
- CD 수령 확인서
- 해시값 확인 스크린샷

## 합격 기준
최종 시험품 CD가 해시값을 포함하여 정상적으로 수령되었다.

## 참조 문서
| 문서명 | 종류 | 설명 |
|--------|------|------|
| 프로그램 | file | 최종 수정본 제품 프로그램 |
| 사용자 매뉴얼 | file | 최종 수정본 사용자 매뉴얼 |
| 제품 설명서 | file | 최종 수정본 제품 설명서 |
| 코드 해시 변환값 | file | 프로그램 개발 코드의 해시값 |

## TIP
> 전달 방법: 업체 직접 방문, 퀵, 소포 등
> 해시값이 누락된 경우 반드시 업체에 재요청하세요.
