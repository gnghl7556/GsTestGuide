## 설명
시험수행 완료 후, 업체로부터 최종 시험품(CD)을 수령하고 내용물의 완전성을 검증하는 단계. 최종 시험품은 인증위원회에 제출되는 공식 시료이므로 프로그램, 매뉴얼, 제품설명서, 해시값이 빠짐없이 포함되어야 한다.

## 체크포인트
- [ ] 업체가 최종 시험품을 CD로 제공했는가? (전달 방법: 직접 방문, 퀵, 소포 등)
- [ ] CD에 프로그램, 사용자 매뉴얼, 제품설명서가 모두 포함되어 있는가? [ref: 프로그램, 사용자 매뉴얼, 제품 설명서]
- [ ] 최초 제공 시험품과 동일한 형태(설치 방식, 파일 구조 등)인가?
- [ ] 프로그램 개발 코드의 해시값이 포함되어 있는가? 누락 시 업체에 즉시 재요청했는가? [ref: 코드 해시 변환값]
- [ ] 시료 반입/반출 확인서를 작성하고 관리 담당자에게 제출했는가?

## 증빙 예시
- 제품 CD (프로그램, 사용자 매뉴얼, 제품설명서, 해시값 포함)
- 시료 반입/반출 확인서 (관리 담당자 제출용)
- CD 내용물 목록 캡처 (파일 구조 확인)

## 합격 기준
최종 시험품 CD가 해시값을 포함하여 정상적으로 수령되었다.

## 참조 문서
| 문서명 | 종류 | 설명 |
|--------|------|------|
| 프로그램 | file | 최종 수정본 제품 프로그램 |
| 사용자 매뉴얼 | file | 최종 수정본 사용자 매뉴얼 |
| 제품 설명서 | file | 최종 수정본 제품 설명서 |
| 코드 해시 변환값 | file | 프로그램 개발 코드의 해시값 |

## TIP
> 해시값이 누락된 경우 반드시 업체에 재요청하세요. 해시값 없이는 인증위 제출이 불가합니다.
> CD 내용물이 패치 후 최종 버전과 일치하는지 확인하세요. 패치 전 버전이 들어있는 경우가 간혹 있습니다.
