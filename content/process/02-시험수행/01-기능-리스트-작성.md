## 설명
시험 환경 구성 완료 후, 사용자 매뉴얼과 제품 설명서를 기반으로 기능 리스트를 작성하고 품질특성별로 분류하는 단계

## 체크포인트
- [ ] 프로그램 및 매뉴얼(또는 제품 설명서)을 기반으로 시험 제품의 주요 기능을 세 분류(대·중·소)로 구분하였는가? [ref: 사용자 매뉴얼, 제품 설명서]
- [ ] '기능 리스트' 양식에 맞게 주요 기능을 구분하고 기능에 대한 설명을 작성하였는가? [ref: 기능리스트(양식)]
- [ ] 업체에서 제공한 '기능 리스트'와 비교하여 누락된 기능이 없는지 확인하였는가? [ref: 기능리스트(업체)]

## 증빙 예시
- 기능리스트

## 합격 기준
기능 리스트가 작성되어 품질특성별 결함 탐색 및 테스트가 가능한 상태이다.

## 참조 문서
| 문서명 | 종류 | 설명 |
|--------|------|------|
| 사용자 매뉴얼 | file | 제품 기능 파악 및 기능 리스트 작성 기준 |
| 제품 설명서 | file | 제품 기능 구조 및 상세 스펙 확인 |
| 기능리스트(업체) | file | 업체 제공 기능 리스트 (참고용) |

## TIP
> 업체 기능 리스트는 테스트 적합성이 낮으므로 참고용으로만 활용하세요.
> 기능 리스트는 시험 종료 시까지 지속적으로 보완합니다.
