## 설명
기본 환경 구성 완료 후, 서버 PC가 포함되는 경우 Putty를 통한 SSH 원격 접속 환경을 추가 구성하는 단계

## 체크포인트
- [ ] 시험에 Linux/CentOS 서버 PC가 포함되는지 확인했는가? [ref: 시험 합의서]
- [ ] (조건부) Putty를 통한 서버PC 접근이 필요한 제품인가?
- [ ] (조건부) 서버 PC에 SSH 접속이 정상적으로 되는가? [ref: 계정 정보(업체)]

## 증빙 예시
- 환경구성(구글 스프레드 시트, PC 및 서버 계정 접근 정보 기입)

## 합격 기준
1. 서버 PC 포함 시 원격 접속 환경이 구성되어 Putty를 통해 서버에 접근이 가능
2. 접근한 서버PC의 계정 정보를 '환경구성'파일에 입력하고, 외부 접근을 방지하기 위해 계정 비밀번호를 변경

## 참조 문서
| 문서명 | 종류 | 설명 |
|--------|------|------|
| 시험 합의서 | file | 서버 PC 포함 여부 및 환경 요구사항 확인 |
| 계정 정보(업체) | file | 서버 PC 접속용 계정 정보 |

## TIP
> Linux/CentOS 서버가 시험에 포함되는 경우가 대부분이므로 Putty 설치를 기본으로 진행하세요.
