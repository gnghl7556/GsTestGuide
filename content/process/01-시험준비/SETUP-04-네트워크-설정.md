## 설명
IP 담당자에게 필요한 IP를 요청받고, TTA 제공 양식에 따라 네트워크를 설정하여 인터넷 연결을 확인한다.

## 체크포인트
- [ ] IP 담당자에게 필요한 IP 개수를 요청하고 수령했는가?
- [ ] TTA 양식에 따라 네트워크 설정을 완료했는가?
- [ ] 인터넷 연결이 정상적으로 작동하는가?

## 증빙 예시
- PL이 공유한 '환경구성'엑셀 파일(or 구글 스프레드시트)에 각 장비에 부여된 IP주소 입력하여 증빙
## 합격 기준
1. 네트워크가 TTA 양식에 맞게 설정되고 인터넷 연결이 정상 작동한다.
2. PL이 공유한 '환경구성'파일에 'IP'정보가 모두 입력

## 참조 정보
| 항목 | 내용 | 링크 |
|------|------|------|
| IP 주소 | (수령받은 IP) | |
| 서브넷 마스크 | 255.255.255.0 | |
| 기본 게이트웨이 | 210.104.181.1 | |
| DNS 서버 | 168.126.63.1 | |
