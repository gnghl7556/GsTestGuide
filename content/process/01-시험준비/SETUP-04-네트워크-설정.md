## 설명
SW 환경 구성 후, IP 담당자로부터 IP를 수령하고 네트워크를 설정하여 인터넷 연결을 확보하는 단계. 서버 PC가 포함된 경우 서버용 IP도 별도로 필요하므로 SETUP-01에서 파악한 장비 구성에 맞춰 IP 개수를 산정한다.

## 체크포인트
- [ ] 클라이언트 PC 및 서버 PC 각각에 필요한 IP 개수를 산정하고, IP 담당자에게 요청·수령했는가?
- [ ] TTA 양식에 따라 각 PC의 네트워크(IP, 서브넷 마스크, 기본 게이트웨이, DNS)를 설정했는가?
- [ ] 모든 PC에서 인터넷 연결이 정상 작동하는지 확인했는가?
- [ ] 설정한 IP·네트워크 정보를 환경구성 시트에 기입했는가? [ref: 환경구성]

## 증빙 예시
- 환경구성 시트 (구글 스프레드 시트, IP·서브넷·게이트웨이·DNS 기입)
- 네트워크 설정 완료 화면 캡처 (ipconfig 결과)
- 인터넷 연결 확인 화면 캡처 (브라우저 접속 또는 ping 결과)
## 합격 기준
1. 네트워크가 TTA 양식에 맞게 설정되고 인터넷 연결이 정상 작동한다.
2. PL이 공유한 '환경구성'파일에 'IP'정보가 모두 입력
